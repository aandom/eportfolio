Project information

  • Category: Web Development
  • Project date: 25 September, 2023
  • Project URL: github

Description

The ZU E-Registration system revolutionizes course enrollment for students at our university, addressing existing dissatisfactions by introducing a fully automated solution. With a focus on user convenience and efficiency, our system utilizes advanced technology, including Genetic Algorithms, to intelligently generate conflict-free class schedules. Administrators benefit from streamlined management of courses, instructors, and classrooms, while students can easily initiate add/drop requests. Built with HTML, CSS, and JavaScript for the frontend, and PHP for the backend, this innovative platform promises to enhance the registration experience, reduce administrative workload, and foster greater satisfaction among users.

Description

The Cub3d project is a tribute to classic games like Wolfenstein 3D, developed using the C programming language and Minilibx library. It tasks developers with crafting a 3D maze game using raycasting techniques. Player is immersed in a labyrinthine environment, navigating through dynamic renderings to reach the exit. With a focus on mastering fundamental programming concepts and implementing dynamic rendering, Cub3d offers hands-on experience in game development and 3D graphics programming.

Description

The ft_irc project is a collaborative endeavor designed to create an Internet Relay Chat (IRC) server and client, emphasizing robustness and scalability. Developed using C++, participants engage in crafting a versatile platform facilitating real-time communication among users. With a focus on implementing networking protocols and handling concurrency, ft_irc provides invaluable experience in network programming and software engineering. Whether honing skills or delving into the intricacies of network architectures, ft_irc offers an immersive journey into the world of distributed systems and communication technologies.
